# cranberries
A node.js app (express.js + next.js) for latipay API transactions

## Run
0. nodejs 7.10 or above
1. make sure `config.js` is correct 
2. `yarn` 
3. `npm run dev` or `node server.js`
4. go to `http://localhost:3000/api-console`
## Test
`npm run test`
